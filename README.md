Requirements:
- Java 25
- Maven

Run the game:
- Write "mvn clean install in the terminal".
- Write "mvn exec:exec".

How to start the microservice:
- Open a new terminal, and cd into the "scoring-service".
- Write "mvn spring-boot:run".
- When it is running, open it at: "http://localhost:8080/score/highscore"
